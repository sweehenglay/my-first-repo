Hello World! This is my first GitHub project.

Learning the Big Three Git commands to add commit and push to GitHub manually! Step C: Run the commands Type these commands into that bottom window one by one, pressing Enter after each: git add . git commit -m "Add learning update to index file" git push Once git push finishes running, go look at your my-first-repo repository on the GitHub website. Refresh the page, and you will see your new line of code is live, and your commit message is displayed right next to the file!
